Floating UI & Overlays

Global Navigation Bar (<nav>)

The main navigation bar is a fixed element pinned to the top of the viewport. It uses heavy glassmorphism (blur: 50px, saturate: 180%). On mobile devices, the desktop links are hidden, and a hamburger icon triggers a vanilla JS script that expands the container to reveal the mobile link blocks. Look at the top of this page to see it in action!

Toast Notifications (.toast)

A non-intrusive notification pill that appears fixed at the bottom center of the screen to provide temporary feedback (e.g., when a user clicks "Copy"). It automatically fades out after a set timeout. It is positioned slightly above the scroll button to prevent overlap. Try copying something to see what it looks like.

Scroll To Top Button (#scrollTop)

A circular glassmorphic button that slides up from the bottom center of the screen once the user scrolls down past a predefined threshold (e.g. 300px). Clicking it triggers smooth scrolling back to the top of the document. It’s intended for overly long web pages, like this one.
